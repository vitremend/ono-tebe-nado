# Проект: Оно тебе надо

### Обзор
Учебный проект по верстке сайта-аукциона в рамках курса Яндекс Практикум. Работа выполнена с использованием семантической разметки HTML5 и методологии БЭМ.

### Ссылки
*   Ссылка на репозиторий: https://github.com/vitremend/ono-tebe-nado
*   Ссылка на GitHub Pages: https://github.io

### Технологии
* HTML5
* CSS3 (Flexbox)
* Методология БЭМ
* Адаптивная верстка
